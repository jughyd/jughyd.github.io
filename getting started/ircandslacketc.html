Apart from the meetup site, which we use to schedule meetups, we use various other channels for the purpose of communicating with the group.

IRC

Internet Relay Chat (IRC) is an application layer protocol that facilitates transfer of messages in the form of text. IRC clients are computer programs that a user can install on their system. These clients are able to communicate with chat servers to transfer messages to other clients. It is mainly designed for group communication in discussion forums, called channels, but also allows one-to-one communication via private message.
— Wikipedia

Usage Instructions

Once you are inside IRC client, run the following commands.

nick <yourname> (1)
server irc.freenode.net (2)
msg nickserv register <password> <email> (3)
msg nickserv identify <password> (4)
join #jughyd (5)
<1> For chatting in IRC. Your proper name is recommended. No funny names.
<2> Opens the irc server
<3> This registers the previously provided name for you. This command is only required once in a life time of your user name. Once succesfully registered, you never have to register again.
<4> To login to the server. Make sure you have set /nick properly.
<5> Enter into jughyd channel.
